Le protocole HTTP

Quoi ?

Est comme un facteur qui livre les pages...

...Web à votre maison (ordinateur).

C’est un protocole qui permet à votre ordinateur...

Le protocole HTTP

Quoi ?

...de parler aux serveurs Web.





Le protocole HTTP

Qui ?

Tout le monde qui utilise un navigateur Web utilise HTTP.

Les développeurs Web l’utilisent pour créer des sites Web.

Les serveurs Web l’utilisent pour répondre aux demandes des utilisateurs.

Le protocole HTTP

Où ?

HTTP est utilisé partout sur Internet.

Dans les navigateurs Web sur votre ordinateur ou votre téléphone.

Les serveurs Web l’utilisent pour envoyer des pages Web à votre navigateur.

Le protocole HTTP

Quand ?

HTTP est utilisé chaque fois que vous visitez un site Web.

Lorsque votre navigateur demande une page Web à un serveur.

Lorsque vous cliquez sur un lien ou soumettez un formulaire sur un site Web.

Le protocole HTTP

Comment ?

Votre navigateur envoie une demande HTTP à...

...un serveur, comme demander une lettre.

Le serveur répond avec une réponse HTTP, comme...

Le protocole HTTP

Comment ?

...recevoir une lettre.

Votre navigateur lit la réponse et affiche...

...la page Web.

Le protocole HTTP

Combien ?

Il existe deux versions principales de HTTP :...

...HTTP/1.1 et HTTP/2.

HTTP/1.1 est la version la plus couramment utilisée aujourd’hui.

Le protocole HTTP

Combien ?

HTTP/2 est une version plus récente et plus rapide de HTTP.





Le protocole HTTP

Pourquoi ?

Parce qu’il est simple et universellement pris en charge.

Pour permettre aux ordinateurs de...

...communiquer efficacement sur Internet.

Le protocole HTTP

Pourquoi ?

Sans HTTP, nous ne pourrions pas naviguer sur...

...le Web comme nous le faisons aujourd’hui.
